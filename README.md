# django-boilerplate
Django ecommerce.


# Getting Started

First Create and start your virtual environment:
```
  pip install -r requirement.txt
  python manage.py makemigrations
  python manage.py migrate
  python manage.py createsuperuser
  python manage.py runserver
```
